# Self-uninstallation removes managed state

`sklp uninstall` is a confirmation-based lifecycle command for macOS, Linux, and Windows. After the user enters `y`, it removes the verified bundled Agent integration, managed Agent entries recorded by Hub state, the Hub and its managed Skills, the Hub locator, and the npm-global `skill-port-cli` package. It checks only the deterministic bootstrap path and Hub-recorded entries; it does not scan the filesystem. Hub-external linked Skill sources, conflicting unmanaged entries, and the source checkout remain untouched. Bootstrap ownership follows [ADR 0004](0004-global-install-bootstraps-agent-integration.md).
