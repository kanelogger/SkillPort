## Windows Evidence Policy

Windows support is closed by hosted GitHub Actions `windows-latest` evidence. The release candidate must pass CI before tag/release, and the published package must pass the post-publish install smoke after npm publication.

## User Runtime Promise

For local Skill lifecycle commands, Windows users need Node.js `>=24.15.0` and npm. Git-source commands additionally require a system `git` executable on `PATH`.
